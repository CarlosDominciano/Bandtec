Listas

Listas são usadas para listar conteúdos de forma ordenada, não ordenada ou por definição.

Listas ordenadas

Listas ordenadas são feitas quando a sequência de cada item é relevante para o entendimento, como uma sequências de passos para realizar alguma ação

O fluxo do teste de acessibilidade pode se encaixar em uma lista ordenada

  1. Realizar modelagem
  2. Realizar teste e retestes
  3. Gerar parecer

Listas não ordenadas

Na lista não ordenada, temos as Rts que atendemos em investimentos

  • rt canais atuais
  • RT novos canais
  • RT assessoria

Listas de definição

Listas de definição são usadas para componentes onde existe um termo e em seguida sua definição, como se fosse pergunta e resposta.

Podemos usar com os termos de WCAG e W3C.

WCAG
"Web Content Accessibility Guidelines", recomendações de acessibilidade para conteúdo da web
W3C
"World Wide Web Consortium", consórcio internacional com o objetivo de desenvolver padrões para web

Links

Links são usados para realizar conexão dentro da própria página, localmente e globalmente.

Links internos

Ao clicar em um link interno, você será direcionado para algum lugar dentro da própria página.

Links internos geralmente são usados para avançar blocos de conteúdos, os links âncoras, eles também podem ser usados para voltar para o topo da tela.

Links locais

Ao clicar em um link local, você será direcionado para uma página dentro do mesmo site.

Links externos

Ao clicar em links externos, você vai ser direcionado para um outro site.

No exemplo, o google vai ser aberto em uma nova aba, por isso a orientação no rótulo do link.
